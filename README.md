# Game 2048
## Giới thiệu
Đây là đồ án game 2048 được xây dựng bằng ngôn ngữ python cùng với thư viện pygame

## Cài đặt môi trường & chạy chương trình
Khuyển khích cài đặt micromamba để thuận tiện hơn trong việc cài đặt môi trường và dự án
### 1 Cài đặt môi trường
```python
micromamba create -n game_2048 python=3.9
micromamba activate game_2048
```
### 2 Cài dặt dự án
```python
git clone https://github.com/Qtt2101/2048GameProject ##Clone git về
pip install -r requirements.txt ##Cài đặt các thư viện cần thiết
python main.py ##Bắt đầu trò chơi
```
