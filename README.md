Đây là dự án game 2048 được xây dựng bằng Python và thư viện Pygame.

Yêu cầu hệ thống:

-Python 3.10.19 trở lên

-pygame 2.5.2 trở lên

-numpy 1.26.4 trở lên

-torch 2.3.0 trở lên

Cách chạy:

-Mở terminal để di chuyển đến thư mục project (hoặc mở thư mục chứa project bằng các IDE như Visual Studio Code,...)

-Chạy lệnh:

python main.py
